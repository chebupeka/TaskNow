# TaskNow
TaskNow — это веб приложение, автоматизирующее процесс срочного подбора временных работников для малого оффлайн бизнеса.  Проект вдохновлён моделью Yandex Go и переносит её принципы на рынок временного труда: вместо заказа поездки пользователь создаёт задачу, а система автоматически подбирает подходящего исполнителя.
